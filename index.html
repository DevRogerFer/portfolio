Sobre Mim

Sou Desenvolvedor Full Stack, graduado em Engenharia de Software e Pós-graduado em Desenvolvimento Web Full Stack, com experiência no desenvolvimento de soluções digitais sob medida para empresas, startups e profissionais que buscam eficiência, qualidade e inovação. Atuo na análise e avaliação de projetos de software, transformando ideias e requisitos em soluções práticas, escaláveis e bem estruturadas. Tenho facilidade em entender necessidades de negócio e traduzi-las em aplicações funcionais, seguras e de alto desempenho. Trabalho com tecnologias modernas como Python, Django, Java, JavaScript, TypeScript, React, Node.js e Next.js, desenvolvendo aplicações web completas — do back-end ao front-end — sempre seguindo boas práticas de código e arquitetura. Como freelancer, meu foco é entregar valor real, cumprir prazos e manter uma comunicação clara durante todo o projeto. Estou aberto a novos desafios, parcerias e projetos que exijam soluções bem pensadas e resultados concretos.

Projetos

Projeto Psique - Assistência Psicológica

Psiquê é uma aplicação web desenvolvida em Django para apoiar profissionais de psicologia na gestão de pacientes, registro de gravações de sessões, transcrição automática de áudio/vídeo, geração de resumos e avaliação de humor com apoio de modelos de IA. O sistema também treina uma base vetorial (RAG) por paciente para permitir consultas conversacionais sobre o conteúdo das sessões.

Projeto Pizzaria - Gestão de Produtos e Pedidos

Desenvolvimento de uma aplicação completa para atendimento de uma pizzaria, contemplando a criação do backend com a arquitetura do servidor, implementação das regras de negócio e construção de uma API robusta e escalável. Criação do frontend englobando interfaces intuitivas, dinâmicas e alinhadas às necessidades do usuário. Por fim, desenvolvimento do aplicativo Mobile integrando-se à API do sistema, garantindo uma comunicação eficiente e proporcionando uma experiência fluida, responsiva e otimizada para dispositivos móveis.

Projeto Chatbot de Programação - Interativo Básico

Um chatbot interativo básico desenvolvido com React.js que responde perguntas sobre programação usando um sistema de correspondência de palavras-chave e um conjunto fixo de respostas pré-definidas. Desenvolvido como "Projeto 5 Estrelas" para o programa de treinamento The Accelerator da plataforma Workana.

Projeto Machine Learning - Previsão de Dados

Um projeto de Machine Learning desenvolvido para prever dados com base em padrões históricos. Utilizando técnicas avançadas de aprendizado supervisionado e não supervisionado, o sistema é capaz de gerar previsões precisas e insights valiosos para a tomada de decisões.
